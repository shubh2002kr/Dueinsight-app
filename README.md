## ⚡ Run Locally

### 1️⃣ Clone Repository
```bash
git clone https://github.com/shubh2002kr/Dueinsight-app.git
cd Dueinsight-app
```

### 2️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 3️⃣ Run Streamlit App
```bash
streamlit run app.py
```
